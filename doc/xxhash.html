hash32/1

hash32(Data::hash_input()) -> hash_digest()

hash32/2

hash32(Data::hash_input(), Seed::hash_seed()) -> hash_digest()

Hash Data with Seed value, and return the digest.

See also: hash32/1.

hash32_digest/1

hash32_digest(Handle::hash_handle()) -> hash_digest()

Calculates an intermediate digest of all the passed data to the Handle hasher. Unlike hash32_final/1, the hasher can still be used after hash32_digest/1 has been called.

hash32_final/1

hash32_final(Handle::hash_handle()) -> hash_digest()

Calculates the final hash of all the passed data to the Handle hasher. Unlike hash32_digest/1, the hasher is destroyed and can't be used after hash32_final/1 has been called.

hash32_init/0

hash32_init/1

hash32_update/2

hash32_update(Handle::hash_handle(), Data::hash_input()) -> ok

Update the Handle hasher content with the given Data. This can be called many times with new data as it is streamed.
